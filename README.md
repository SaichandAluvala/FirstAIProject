## Installation

To run the programs in this repository, you'll need Python and the following libraries installed:

- `numpy`
- `scikit-learn`
- `tensorflow`
- `matplotlib`

You can install these dependencies using pip:

```sh
pip install numpy scikit-learn tensorflow matplotlib
